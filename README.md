# Dami Wamisalami ECE ePortfolio

This repository contains a published ePortfolio for the ECE ePortfolio assignment.

## Features

- Home, About, Career Goals, Resume, Technical Projects, Contact sections
- 1,000+ words of text
- 5+ images used from free sources
- Downloadable `assets/Resume.pdf`
- Smooth scrolling navigation
- Minimalist design following modern usability standards (Apple/Google/Meta)

## How to publish

1. Create a branch (optional): `git checkout -b main`
2. Commit all changes: `git add . && git commit -m "Initial ePortfolio site"`
3. Push: `git push origin main`
4. In GitHub repository settings, enable **GitHub Pages** on branch `main` and folder `/`.
5. Share your live link in Canvas for the assignment.

## Notes

- Replace `mailto:dami@example.com` with your real email.
- Replace LinkedIn and GitHub links with your personal profiles.
- Replace `assets/Resume.pdf` with your actual resume PDF from your design.
